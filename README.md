# IITK Mini MIPS Processor
Designed and implemented a 32-bit MIPS-like processor in Verilog with support for R-, I-, and J-type instructions, and deployed it on a Xilinx Spartan-3E FPGA as part of a 2-member team.

# Project Repository
[APOLCLP/IITK_-mini-mips](https://github.com/APOLCLP/IITK_-mini-mips)
